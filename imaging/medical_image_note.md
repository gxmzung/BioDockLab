# Medical Imaging / Surgery AI Direction

This module is reserved for future Surgery AI research.

## Planned Stack

- OpenCV
- DICOM
- PyTorch
- MONAI

## Initial Goal

BioDockLab will not directly implement surgical AI at the first stage.

The first goal is to describe the data flow:

Medical Image → Feature Extraction → Risk Estimation → Decision Support → Report
