# BioDockLab

Web-based Protein-Ligand Docking Visualization Platform

## Overview

BioDockLab is an educational and exploratory platform that visualizes protein-ligand docking workflows using open-source bioinformatics tools and public molecular datasets.

This project combines:
- Bioinformatics
- Artificial Intelligence
- Computer Engineering
- Game Engineering

into a collaborative interdisciplinary platform.

---

## Key Features

- Protein structure visualization
- Ligand docking result rendering
- Interactive 3D molecular viewer
- Docking score comparison
- Browser-based simulation workflow

---

## Tech Stack

### Backend
- Python
- FastAPI

### AI / Docking
- AutoDock Vina
- DeepChem

### Frontend
- HTML/CSS/JavaScript
- WebGL
- NGL Viewer

---

## Honest Limits

BioDockLab is an educational and exploratory platform.

Docking scores are predictive values only and must not be interpreted as clinically validated medical conclusions.
