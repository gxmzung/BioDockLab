# Local Setup

## Purpose

This document explains the local development setup direction for BioDockLab.

## Requirements

- Python
- Node.js
- npm
- Git

## Backend Direction

The backend is based on FastAPI.

Expected workflow:

1. Install Python dependencies.
2. Run the FastAPI backend.
3. Test experiment and analysis endpoints.

## Frontend Direction

The frontend is based on React and TypeScript.

Expected workflow:

1. Install npm dependencies.
2. Run the frontend development server.
3. Connect UI screens to backend API.

## Current Status

Local setup is still being refined.

This document will be updated as the executable workflow becomes more stable.
