# Presentation Flow

## 1. Opening

Introduce BioDockLab as a bio AI research software prototype.

## 2. Problem

Explain that biological experiment data is often scattered across notes, files, and manual reports.

## 3. Solution

BioDockLab connects experiment records, AI-assisted analysis, simulation, and report generation.

## 4. System Flow

Experiment Data → AI Analysis → Digital Twin Simulation → Research Report

## 5. Validation

Mention:

- Grand Prize
- University startup support selection
- Expert feedback
- Follow-up development planning

## 6. Boundary

Clearly state that BioDockLab is not a medical device, diagnostic tool, or clinical decision-support system.

## 7. Roadmap

Explain the MVP direction and future development path.
