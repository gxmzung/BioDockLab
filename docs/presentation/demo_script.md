# BioDockLab Demo Script

## 1. Open Platform
Open the BioDockLab web UI.

## 2. Select Protein
Choose a PDB target such as 6LU7.

## 3. Review Metadata
Show RCSB metadata: title, method, resolution, citation.

## 4. View 3D Structure
Rotate and zoom the WebGL molecular viewer.

## 5. Show Docking Config
Explain docking box center, box size, exhaustiveness, receptor path.

## 6. Create Sample Experiment
Click Create Sample Experiment and show generated experiment ID, score, output JSON, and log file.

## 7. Explain Claim Boundary
State clearly: this is a research/education workflow platform, not clinical validation or actual drug discovery.
