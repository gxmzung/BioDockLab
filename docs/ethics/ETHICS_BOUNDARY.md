# Ethics Boundary

## Purpose

This document defines the ethical boundary of BioDockLab.

BioDockLab is a research software prototype for experiment data management, AI-assisted analysis, simulation, and report generation.

It must not be presented as a medical device, diagnostic tool, or clinical decision-support system.

## Ethical Principles

BioDockLab should follow these principles:

- Do not overclaim medical capability.
- Do not use real sensitive medical data at the current stage.
- Keep analysis logic explainable.
- Clearly document limitations.
- Require expert review for bio-domain claims.
- Separate research software from clinical use.

## In Scope

BioDockLab may explore:

- Research workflow support
- Experiment data organization
- Explainable analysis
- Simulation concepts
- Report automation
- Educational research software

## Out of Scope

BioDockLab does not support:

- Diagnosis
- Treatment recommendation
- Clinical decision-making
- Emergency triage
- Prescription guidance
- Real patient care

## Current Position

BioDockLab should be evaluated as a research-assistant software prototype.
