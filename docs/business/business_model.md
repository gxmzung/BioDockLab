# BioDockLab Business Model

## 1. Education SaaS
Target: university classes, interdisciplinary capstone teams, science clubs.
Revenue: institution license, course package, report export.

## 2. Research Workflow Tool
Target: university labs, undergraduate researchers, early bio startups.
Revenue: lab subscription, setup consulting, self-hosted package.

## 3. No-Code Bio Pipeline Builder
Target: non-programming researchers.
Revenue: pro workspace, pipeline templates, cloud execution credits.

## 4. API Platform
Target: AI/bioinformatics developers.
Revenue: API key, usage-based billing, enterprise deployment.
