# Business Documentation

This directory contains business-oriented documents for BioDockLab.

## Purpose

BioDockLab is primarily a research software prototype, but it also has potential as a startup or university-supported research platform.

## Business Direction

Possible business directions include:

- Bio research assistant platform
- Lab experiment data management tool
- AI-assisted research report generator
- Education tool for bio research workflows
- Research software for university labs

## Current Stage

BioDockLab is not a commercial product yet.

The current focus is:

- Prototype development
- Expert feedback
- Documentation
- MVP validation
- Startup support preparation
