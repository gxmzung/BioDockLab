# Module Map

## Purpose

This document maps the main BioDockLab directories to their responsibilities.

## Core Modules

### backend

FastAPI backend prototype for experiment APIs and analysis endpoints.

### frontend

Dashboard and user interface prototype.

### ai

Experiment analysis, risk classification, and feature engineering.

### simulation

Digital twin-style simulation, organoid response simulation, and CFPS yield simulation.

### reports

Research report generation and export direction.

### sample_data

Demonstration datasets for testing workflow structure.

### docs

Architecture, research, validation, operations, security, and planning documents.

## Supporting Modules

### bio

Bio-domain logic and biological data utilities.

### imaging

Long-term medical and biological imaging research direction.

### quantum

Long-term quantum biocomputing research notes.

### viewer

Data viewer prototype direction.

## Current Status

The module structure is designed to support future team collaboration and gradual MVP integration.
