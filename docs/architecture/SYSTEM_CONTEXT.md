# System Context

## Purpose

This document explains the system context of BioDockLab.

BioDockLab is a research software prototype that connects biological experiment records, AI-based analysis, digital twin-style simulation, dashboard visualization, and report generation.

## Users

Potential users include:

- Researchers
- Student researchers
- Bio-domain collaborators
- Project reviewers
- Technical mentors

## External Inputs

BioDockLab may receive:

- Experiment records
- Sample metadata
- Condition values
- Observation notes
- Success rate values
- Risk indicators

## Main System Boundary

BioDockLab focuses on research workflow support.

It does not provide:

- Diagnosis
- Clinical decision-making
- Medical device functionality
- Validated biological prediction

## Output

BioDockLab may produce:

- Experiment summaries
- Risk analysis results
- Simulation outputs
- Dashboard views
- Research reports

## Current Stage

The current system is an MVP / prototype.
