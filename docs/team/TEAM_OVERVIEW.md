# BioDockLab Team Overview

## 1. Team Direction

BioDockLab은 의료·바이오 데이터를 역할별로 안전하게 보여주고, 환자 설명, 연구자 실험, 보안 감사 로그를 연결하는 웹 기반 프로토타입이다.

초기 팀은 정규 채용 조직이 아니라 **대학동아리 기반 프로젝트팀**으로 운영한다.

팀원에게 제공할 수 있는 가치는 다음과 같다.

- 활동비
- 회의비
- 식비
- 교통비
- 공모전 수상금 배분
- GitHub 기여 기록
- 포트폴리오 산출물
- 발표/공모전 경험
- 의료·바이오 융합 프로젝트 경험
- 창업/사업화 검토 경험

## 7. Data Safety Rule

BioDockLab 팀은 다음 원칙을 지킨다.

1. 실제 환자 데이터 사용 금지
2. 병원 내부 데이터 무단 수집 금지
3. 공개 데이터 또는 샘플 데이터 우선 사용
4. 비식별화 원칙 준수
5. 의료진/교수 자문 전 진단·처방 표현 금지
6. AI 기능은 설명과 상담 준비 보조로 제한
7. 실제 상용화 전 법률·개인정보·보안 검토 필수

## 8. Current Team Assessment

## Strengths

- Product/System 설계 담당이 있음
- AI/의료 데이터 관심 인력이 있음
- 디자인 후보가 있음
- 사업/데이터 분석 후보가 있음
- 경영/회계/의료 비즈니스 관점 후보가 있음

## Weaknesses

- 백엔드/보안 전담이 부족함
- 실제 간호/의료 현장 자문자가 부족함
- 바이오/단백질 도메인 검토자가 부족함
- 법률/개인정보 자문이 없음

## Next Action

1. 백엔드/보안 개발자 후보 찾기
2. 간호학과 또는 간호사 자문자 확보
3. 생명공학/약학/바이오 전공자 자문 확보
4. 팀원별 역할 설명 후 GitHub Project에 배정
5. 활동비 기반 운영 기준 정리
