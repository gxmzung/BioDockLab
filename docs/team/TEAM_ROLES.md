# Team Roles

## Purpose

This document defines possible team roles for BioDockLab.

BioDockLab is a multidisciplinary project that connects computer science, bio research, AI, product planning, and documentation.

## Core Roles

### Project Lead

Responsibilities:

- Product direction
- System architecture
- GitHub management
- Documentation quality
- Presentation and Q&A

### Backend Developer

Responsibilities:

- FastAPI backend
- API route design
- Experiment data handling
- Analysis endpoint integration

### Frontend Developer

Responsibilities:

- Dashboard UI
- Data visualization
- Research workflow screens
- User experience improvement

### AI / Data Developer

Responsibilities:

- Risk classification logic
- Feature engineering
- Experiment analysis
- Simulation support

### Bio-Domain Advisor

Responsibilities:

- Bio workflow review
- Experiment assumption review
- Terminology correction
- Research direction feedback

### Business / Planning Member

Responsibilities:

- Business model
- Market research
- Startup support documents
- Presentation materials

## Current Position

BioDockLab is still in early-stage team formation.

The current priority is to keep roles clear and avoid overclaiming domain expertise.
