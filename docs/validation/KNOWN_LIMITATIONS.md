# Known Limitations

## Technical Limitations

- Frontend and backend are not fully integrated.
- Experiment CRUD is not fully implemented.
- AI analysis is rule-based.
- Digital twin simulation is function-level.
- Report generation is not fully automated.

## Data Limitations

- Real biological validation data is not included.
- Sample datasets are for demonstration.
- Clinical datasets are not used.
- Model performance is not measured.

## Product Limitations

- User roles need more validation.
- Research workflow assumptions need expert review.
- Security model is early-stage.
- Deployment plan is not production-ready.

## Medical Boundary

BioDockLab is not a medical device, diagnostic system, or clinical decision-support system.
