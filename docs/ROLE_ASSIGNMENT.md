# Role Assignment

## Team Role Structure

| Track | Role | Main Tasks | Output |
|---|---|---|---|
| Bioengineering / Chemical Engineering | Domain Analyst | Protein target selection, biological explanation, result review | Target report, interpretation notes |
| Artificial Intelligence | Data / Ranking Lead | Docking score analysis, ligand ranking, optional ML extension | Score table, ranking logic |
| Computer Engineering | System Architect | Backend, API, data pipeline, repository structure | FastAPI server, JSON schema, integration |
| Game Engineering | UX / Visualization Lead | 3D interaction, WebGL viewer, visual effects | Viewer UI, interaction prototype |

## Communication Rules

- All technical decisions must be written in GitHub Issues or meeting notes.
- Each feature should have one owner.
- Experimental results should be stored in `/experiments`.
- Sample files should be stored in `/sample_data` with clear source notes.
