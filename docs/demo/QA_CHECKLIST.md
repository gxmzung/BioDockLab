# BioDockLab QA Checklist

- Login screen appears before platform access.
- Role select is locked after login.
- Patient can access Patient View.
- Patient cannot access Research Lab or Security Center.
- Pharmacist can access prescription review.
- Admin staff sees appointment and consent data with masking.
- Researcher can access Research Lab.
- Security can access Security Center.
- Platform Admin can access Admin Console.
- Unauthorized access creates audit logs.
- Smoke test returns ALL TESTS PASSED.
