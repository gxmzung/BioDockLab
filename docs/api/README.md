# API Documentation

This directory contains API-related documents for BioDockLab.

## Purpose

The API layer connects experiment records, AI analysis, simulation results, and report generation.

BioDockLab currently uses FastAPI as the backend prototype framework.

## Documents

- API Specification
- Research API Specification

## API Scope

Current API direction includes:

- Experiment list
- Experiment detail
- AI analysis result
- Risk classification
- Digital twin simulation
- Report generation

## Current Status

The API is in prototype stage.

It is not production-ready and does not include finalized authentication, authorization, or regulated medical data handling.
