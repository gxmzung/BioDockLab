# Nurse Workspace 검토 문서

## 목적

간호사 역할은 BioDockLab에서 새로 추가된 현장 역할입니다.

간호사는 환자와 가장 가까운 위치에서 바이탈, 투약, 상태 변화, 인수인계를 다룹니다.

따라서 간호사 화면은 의사 화면이나 원무 화면과 달라야 합니다.

## 현재 화면 가정

Nurse Workspace는 다음 정보를 보여줍니다.

- 환자 상태 요약
- 바이탈 요약
- 투약 전 확인
- 인수인계 요약
- 환자 동의 상태
- 민감정보 최소 표시
- 접근 로그 기록

## 간호사 화면에서 필요한 정보 가설

| 정보 | 이유 |
|---|---|
| 바이탈 요약 | 환자 상태 변화 확인 |
| 최근 처치/투약 확인 | 교대 및 투약 전 확인 |
| 인수인계 요약 | 간호사 간 업무 연속성 |
| 주의 알림 | 낙상, 알레르기, 감염, 상태 악화 가능성 |
| 환자 설명 상태 | 환자가 어떤 설명을 받았는지 확인 |
| 동의 상태 | 리포트/연구 활용 관련 확인 |

## 넣으면 위험할 수 있는 정보

- 의사의 최종 진단 판단을 대신하는 표현
- 처방 변경 제안
- 약물 효과 단정 표현
- 연구자용 원천 데이터
- 불필요한 유전자 상세
- 보안 로그 수정 권한

## 교수/전문가에게 확인할 질문

1. 간호사 화면에 바이탈 요약을 넣는 것이 적절한가?
2. 인수인계 요약 자동화가 현장에서 도움이 될 수 있는가?
3. 간호사가 환자용 설명 리포트를 볼 수 있어야 하는가?
4. 투약 전 확인 정보는 약사 화면과 어떻게 구분되어야 하는가?
5. 간호기록/EMR 대체처럼 보이지 않게 하려면 어떤 문구가 필요한가?
