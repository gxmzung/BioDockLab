# BioDockLab 취창업동아리 / 뉴스 소개 문구

## 안전한 한 줄 소개

BioDockLab은 환자·간호사·의사·약사·원무·연구자·보안관리자가 각자 필요한 의료·바이오 정보를 안전하게 확인할 수 있도록 역할별 화면, 접근 권한, 감사 로그를 구현한 웹 기반 프로토타입입니다.

## 보도자료용 짧은 설명

컴퓨터공학 전공 학생이 개발한 BioDockLab은 의료·바이오 데이터가 사용자 역할에 따라 다르게 보여야 한다는 문제의식에서 출발했다. 이 프로젝트는 환자에게는 쉬운 설명 리포트를, 간호사에게는 바이탈과 인수인계 보조 정보를, 약사에게는 처방량과 약물상호작용 검토 정보를, 연구자에게는 단백질-화합물 가상 도킹 실험 화면을 제공한다. 또한 모든 접근 시도를 감사 로그로 기록하고 권한 없는 접근은 차단하는 보안 구조를 함께 구현했다.

## 강조할 점

- 의료진을 대체하는 AI가 아니다.
- EMR을 대체하는 시스템이 아니다.
- 진단/처방 시스템이 아니다.
- 역할 기반 접근제어와 설명 보조에 초점을 둔다.
- 보안 감사와 개인정보 보호를 핵심 문제로 다룬다.
- 바이오그래머라는 새로운 융합 직무 가능성을 보여준다.

## 피해야 할 표현

- AI가 신약을 개발한다
- AI가 진단한다
- 병원을 대체한다
- EMR보다 좋다
- 처방을 추천한다
- 임상적으로 검증됐다
- 실제 환자 치료에 바로 쓸 수 있다

## 추천 표현

- 프로토타입
- 역할 기반 접근제어
- 설명 보조
- 연구 워크플로우
- 보안 감사
- 의료·바이오 데이터 시각화
- 현장 검토가 필요한 초기 모델
