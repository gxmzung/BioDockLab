# BioDockLab Protein Atlas 확장기획

## 1. 기능 정의

BioDockLab Protein Atlas는 환자 진료용 기능이 아니라, 학생·연구자·비전공자가 단백질 구조, 주요 변이, 관련 질환, 후보 리간드 정보를 한 화면에서 이해할 수 있도록 돕는 교육·연구 보조 기능입니다.

현재 BioDockLab의 핵심이 역할 기반 의료·바이오 데이터 접근제어와 설명 보조라면, Protein Atlas는 연구자·교육용 확장 방향에 해당합니다.

## 2. 왜 필요한가

의료·바이오 데이터는 일반 사용자가 이해하기 어렵습니다. 특히 단백질, 변이, 리간드, 결합 친화도, 도킹 점수 같은 개념은 전공자가 아니면 구조적으로 파악하기 어렵습니다.

Protein Atlas는 다음 문제를 해결하기 위한 화면입니다.

- 단백질 구조와 기능을 한눈에 이해하기 어렵다.
- 특정 변이가 어떤 질환과 관련되는지 파악하기 어렵다.
- 후보 화합물이나 리간드가 어떤 부위에 결합하는지 설명하기 어렵다.
- 연구자 화면과 환자 설명 화면 사이의 교육적 연결이 부족하다.

## 3. 핵심 기능

### 1) 단백질 기본 정보 카드

- 단백질명
- PDB ID
- 주요 기능
- 관련 질환
- 구조 유형
- 주요 변이

예시 단백질:

- EGFR L858R
- KRAS G12D
- BRAF V600E

### 2) 3D 구조 미리보기

- 단백질 표면
- 활성 부위
- 결합 리간드
- 변이 위치 강조

### 3) 질환·변이·리간드 연결

단백질을 중심으로 다음 정보를 연결합니다.

단백질 → 주요 변이 → 관련 질환 → 후보 리간드/화합물 → 연구 실험

### 4) 용어 설명

비전공자가 이해하기 어려운 용어를 함께 설명합니다.

- 활성 부위
- 리간드
- 도킹
- 결합 친화도
- RMSD
- 변이
- 표적치료
- 비식별 데이터

### 5) Research Lab 연동

Protein Atlas에서 선택한 단백질은 Research Lab의 가상 도킹 실험으로 연결될 수 있습니다.

예시 흐름:

EGFR L858R 선택 → 관련 질환 확인 → 활성 부위 확인 → 후보 화합물 선택 → Research Lab에서 도킹 실험 실행

## 4. 추천 샘플 단백질

- EGFR L858R: 폐암 표적치료 예시
- KRAS G12D: 암 표적 연구 예시
- BRAF V600E: 변이 기반 표적치료 설명
- ALK: 융합유전자 및 표적치료 예시
- HER2: 유방암/위암 관련 표적 예시
- SARS-CoV-2 Main Protease: 감염병·항바이러스제 연구 예시

## 5. 활용 가능성

Protein Atlas는 다음 상황에서 활용될 수 있습니다.

- 간호학과·생명과학·보건계열 학생 교육
- 바이오그래머 직무 설명 자료
- 연구자용 실험 전 사전 탐색 화면
- 환자 설명 리포트의 배경 이해 자료
- 단백질-화합물 도킹 실험 흐름의 시각자료

## 6. 주의할 점

이 기능은 실제 진단, 처방, 치료 결정 기능이 아닙니다.

Protein Atlas는 교육·연구 보조 화면이며, 실제 의료 판단은 의료진이 수행해야 합니다. 또한 단백질 구조나 도킹 점수는 실제 임상 효능을 보장하지 않습니다.

## 7. 한 줄 요약

BioDockLab Protein Atlas는 단백질 구조·변이·질환·후보 화합물 정보를 연결해 보여주는 교육·연구용 단백질 도감 기능입니다.
