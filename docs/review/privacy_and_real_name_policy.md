# BioDockLab 실명 사용 및 외부 공개 기준

## 원칙

BioDockLab 화면, 문서, GitHub, 발표자료, 뉴스/홍보자료에는 실제 학생·지인·의료계 종사자의 이름을 사용하지 않는다.

## 이유

실제 인물 이름이 들어가면 다음 문제가 생길 수 있다.

- 본인이 원하지 않는 방식으로 외부에 노출될 수 있음
- 특정 학과/학번/직무와 연결되어 오해될 수 있음
- 의료·간호 관련 프로젝트 특성상 전문직 역할처럼 보일 수 있음
- 뉴스, 교수님 검토, GitHub 자료에 남으면 삭제가 어려움

## 사용 가능한 표현

- 간호학과 검토자 A
- Nurse Demo User
- 데모 간호사
- 김○○
- 샘플 사용자

## 실명 사용이 가능한 경우

다음 조건을 모두 만족해야 한다.

1. 당사자가 명시적으로 동의함
2. 어디에 사용되는지 설명받음
3. GitHub, 발표자료, 뉴스 가능성을 이해함
4. 원하면 언제든 삭제할 수 있음
5. 기여자 표시와 화면 속 인물 표시를 구분함

## 현재 적용

Nurse Workspace의 데모 이름은 `간호학과 검토자 A`로 통일한다.


## 데모 이름 고정 규칙

- `최하늘 원무`는 원무/Administration 역할 전용 데모 이름으로만 사용한다.
- 간호사 역할은 `간호학과 검토자 A`로 익명 처리한다.
- 실제 지인, 학생, 학번이 특정될 수 있는 이름은 외부 공개 자료에 사용하지 않는다.
