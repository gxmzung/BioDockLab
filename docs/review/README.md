# Review Documents

This directory contains review materials for BioDockLab.

## Purpose

BioDockLab requires feedback from bio-domain experts, nursing/medical-related advisors, software mentors, and startup reviewers.

This directory organizes materials for external review.

## Review Focus

Reviewers may check:

- Product concept
- Bio-domain assumptions
- Medical claim boundary
- User role design
- Privacy and data policy
- Research workflow usefulness
- Presentation clarity

## Documents

- Professor review packet
- Clinical assumptions to verify
- Review questions
- Privacy and real-name policy
- Nursing workspace review
- Email attachment checklist
