# BioDockLab Clinical Security Layer v0.1

BioDockLab is not an EMR replacement. It is a clinical explanation, research workflow, and security-audit layer.

Implemented:
- Role-based screens: patient, doctor, pharmacist, administration, researcher, security
- Different visibility per role
- Access-check API
- Audit log generation
- Unauthorized access block message
- Security center audit log review
- Insider over-access/risk event display

Claim boundary: no diagnosis, no prescription, no medical judgment replacement.
