# BioDockLab Product Memory

## What to remember

BioDockLab is now a proprietary/private project focused on commercialization and possible patent exploration.

The product direction is:
1. Role-based access control for bio/clinical data
2. Patient/doctor/pharmacist/admin/researcher/security separated screens
3. Biogrammer job-role separation
4. Audit logs for every access
5. Unauthorized access blocking
6. Insider over-access/risk-event detection
7. Research/education/explanation support only

## Important boundary

BioDockLab is not:
- an EMR replacement
- a diagnosis system
- a prescription system
- a clinical decision-making system
- a validated drug-discovery system

BioDockLab is:
- an explanation support layer
- a research workflow platform
- a security/audit layer
- a bio-data visualization and management system
